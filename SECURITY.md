# Security Policy

## Supported Versions

Only the latest version on the `main` branch is supported.

LGC Concept AI is currently in active development.
Security updates are applied directly to the latest version.

---

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly:

- Do NOT open a public issue.
- Email: lingarobotics@gmail.com

Include:
- Description of the issue
- Steps to reproduce
- Potential impact

Security reports will be reviewed and addressed as appropriate.

---

## Philosophy

Security is treated as an engineering responsibility.
Clarity and safe design take priority over rapid feature expansion.
